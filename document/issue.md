# 需求

需求主要包含:

1. 动态菜单加载(by -zw )
2. 底图加载
3. webgis服务加载(复用搜救系统分离出来的webgis自动化发布功能)
4. 加载海啸站位数据
5. 站位定位检索功能
6. 加载指定历史过程数据(包含计算方法)

## 采用的技术栈

前端:

* vue
* vuex
* ts
* 其他

后端:

* springboot
* jpa

## 1 菜单
实现菜单的动态加载，主要包含以下功能:

* 动态加载父级菜单及子级菜单
* 包含icon,sort等信息
* 包含跳转的url地址信息

建议之后可改进为使用 **SpringSecurity**
